# assignment3
Game Idea

A simple console-based Rock–Paper–Scissors game where you play against the computer. The computer picks randomly, and you decide whether to keep playing.

Rules

0 beats 2 (Rock > Scissors)

1 beats 0 (Paper > Rock)

2 beats 1 (Scissors > Paper)

Same choices → Draw


Controls

0 = Rock

1 = Paper

2 = Scissors

y = Play again

n = Exit
